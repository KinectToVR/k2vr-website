Using Kinect Studio with Xbox One Kinect

The Kinect for Windows SDK that's installed by K2EX Installer comes with an app called Kinect Studio v2.0.

That app lets you see the infrared and depth views of the Kinect as well as the skeleton tracking.

You can open it by searching for studio in the Start menu. The app is named "Kinect Studio v2.0".

By default, you might have to click on "Connect" and/or "Start Monitoring" before you see any output from the Kinect.

Unlike with Kinect Explorer for 360, you can open this app while KinectToVR is running.
