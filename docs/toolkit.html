Using Kinect Explorer with Xbox 360 Kinect

You should close KinectToVR before launching another Kinect app because only one app can access the SDK at once.

While not required by default, and not downloaded by K2EX Installer, because it's almost 400 megabytes in size, the Kinect for Windows Developer Toolkit v1.8 is a supplement to the Kinect for Windows SDK that adds a bunch of really useful sample applications that you can use with the Xbox 360 Kinect and Kinect for Windows V1.

You can download the developer toolkit here: http://download.microsoft.com/download/D/0/6/D061A21C-3AF3-4571-8560-4010E96F0BC8/KinectDeveloperToolkit-v1.8.0-Setup.exe

Launching Kinect Explorer

Once you've installed the Kinect Developer Toolkit, launch the toolkit browser from the Start menu.

Either search for developer or look for it in your recently installed apps list if you just installed it.

Once the app is launched, you can find Kinect Explorer in the Tools tab on the right.

Click "Run" on either Kinect-Explorer-D2D or Kinect-Explorer-WPF to launch them.

The D2D and WPF versions of Kinect-Explorer have the same functionality. It's just a different graphical interface.

Here you have a split view of the color camera which also shows skeleton tracking over it. And in the corner is the depth data created from the infrared camera and dot projector.

You can change the resolution or swap the camera views, and on the right you can also change the Kinect's angle using the tilt motor instead of forcing it manually.

It's possible that these apps won't stop properly when closed and will keep a zombie process running, if KinectToVR doesn't see the Kinect skeleton and you used and closed the app before, verify that it's not still running in Task Manager.
