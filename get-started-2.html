Device compatibility

Webcam

INCOMPATIBLE! Off-the-shelf webcams cannot work with KinectToVR because they lack the additional depth-sensing hardware found in a Kinect.

You should check out AprilTags or Mediapipe instead.


Xbox 360 Kinect / Kinect for Windows V1

This Kinect model is recommended. All features of K2EX will work with it. But it may not fit for smaller rooms. We can help you figure this out over on Discord


Xbox One Kinect / Kinect for Windows V2

While fully compatible, you may run into tracking instability and a have a hard time getting the device to work properly.

We recommend against buying the Xbox One Kinect for reasons stated on --> this page

PlayStation Move + PlayStation Eye

System resource usage is much higher. The setup is long and complicated, and you will need 3 PS Move controllers and at least 2 PS Eye cameras.
Only do this if you already own all the hardware.
